Curso de base de datos con SQLserver

El siguiente curso corresponde a un curso introductorio del manejo de bases de datos con SQL server. El único requisito es contar con equipo de cómputo para aprovechar mejor el curso pues se deberá instalar el gestor de bases de datos SQL server. El curso consta de seis módulos, donde los módulos pueden ser tomados por cualquier persona que desee aprender este lenguaje de programación, sin embargo, el primer módulo contiene teoría que requiere de nociones básicas de Matemáticas formales las cuales comienzan a darse en los primeros semestres de universidad. Aún así los demás módulos serán abordados sin usar lo visto en el primero, pero para entender el curso de manera más general se recomienda estudiar el primer módulo intentando sólo extraer las ideas básicas que se presentan y no la teoría detrás de ellas.
El curso constará de muchas prácticas, ejercicios y ejercicios resueltos. Todo el material que ocupemos podrá descargarse del repositorio del curso en Github, ahí se subirá el código de la prácticas o ejercicios y las bases de datos que ocuparemos. Finalmente, te comento que el curso está organizado por módulos y clases pero también lo tenemos agrupado en un libro, de modo que puedes seguir el curso en cualquiera de las dos alternativas.

Cualquier duda que pueda presentarse referente a este curso puedes mandar DM a nuestra página de Facebook Mat_apuntes, no olvides dejar tu me gusta 👍. ¡Sin más vamos al curso!
